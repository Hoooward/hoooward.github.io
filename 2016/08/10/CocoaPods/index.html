CocoaPods 无法正常使用 pod install

Aug 10, 2016

报错信息: CocoaPods | Unable to add a source with url https://github.com/CocoaPods/Specs.git named master.
  1. 进入 CocoaPods 的目录
    cd ~/.cocoapods/repos
  2. 克隆一份代码下来
    git clone https://github.com/CocoaPods/Specs.git master

关于这个问题的在 github 上的讨论连接:
https://github.com/CocoaPods/CocoaPods/issues/4293
